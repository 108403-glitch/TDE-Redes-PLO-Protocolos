# Estudo Aplicado de Protocolos no Projeto Integrador PLO

Este repositório apresenta o TDE individual da disciplina de Redes de Computadores e Telecomunicações.

O trabalho tem como objetivo analisar a aplicação prática de protocolos e tecnologias de comunicação no Projeto Integrador PLO – Perfil Laser Orçamentos, um sistema web desenvolvido para gerenciamento de solicitações e orçamentos da empresa Perfil Metais.

## Protocolos e tecnologias analisadas

- TCP/IP
- HTTP
- JSON
- API REST
- Arquitetura cliente-servidor
- Comunicação entre front-end, back-end e banco de dados

## Aplicação no Projeto Integrador PLO

A análise relaciona o funcionamento do sistema PLO com conceitos estudados na disciplina, demonstrando como o navegador realiza requisições ao servidor, como os dados são transportados pela rede e como o back-end processa as informações e retorna respostas ao front-end.
